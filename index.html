GPU Path Tracer

  • CUDA
  • C++
  • glTF Loading
  • Open Intel Denoising

A two week solo project. This is a Path Tracer written in C++ and CUDA which utilizes GPU hardware. This is an interactive renderer that interops with OpenGL. It uses fundamental BSDF models such as diffuse, perfect specular, specular transmission and Cook-Torrance microfacet model. It supports glTF scene loading and has Open Intel Denoising Integration.

Part of CIS 5650: GPU Programming.

OpenUSD Asset Database

  • OpenUSD API
  • Houdini Integration
  • postgreSQL Database
  • NodeJS (Frontend)
  • Python (Backend)

A three week class project. This is a deployable USD asset database for an animation/VFX studio. When a USD asset is uploaded by an artist, this app automatically extracts the metadata (i.e. asset name, artist name, versions) and uploads this data into a postgreSQL database. Houdini integration allowed artists to download their assets from the database and continue working on them, allowing them to upload new versions, and even revert back to old versions.

This was an exercise in planning, building and designing a USD pipeline. My contribution to the project was a custom Asset URI Resolver for Houdini, which allowed artists to directly reference assets from the studio database rather than having to download them first. FK/IK Animation System From Scratch

  • C++ & OpenGL
  • Limb-based IK
  • Coordinate Cyclic Descent IK

A 1.5 week solo project. This C++ Unity plugin supports BVH animations to play FK-driven character animation with automatic IK-driven foot placement on uneven terrain. Additionally, characters can be posed by manipulating joints using either IK or FK manipulators.

FK and IK systems are implemented from scratch, including fundamental IK approaches such as Psuedo-Inverse IK, Limb-Based IK & Coordinate Cyclic Descent (CCD) IK.
